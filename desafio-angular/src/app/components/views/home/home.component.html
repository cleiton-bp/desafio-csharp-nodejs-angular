Parte 1: Banco de Dados SQL

  • ◽ Criar um banco de dados simples com duas tabelas relacionadas (por exemplo, "Produtos" e "Categorias").
  • ◽ Inserir alguns dados de exemplo.
  • ◽ Escrever consultas SQL para recuperar informações específicas, como todos os produtos de uma determinada categoria.

Parte 2: API em .NET

  • ◽ Criar uma API REST em .NET Core para expor os dados do banco de dados.
  • ◽ Implementar endpoints para operações CRUD (Criar, Ler, Atualizar, Excluir) em produtos e categorias.
  • ◽ Documentar a API usando Swagger ou ferramenta similar.

Parte 3: API em Node.js

  • ◽ Criar uma API REST em Node.js para consumir a API .NET desenvolvida anteriormente.
  • ◽ Implementar endpoints para chamar as operações CRUD da API .NET.
  • ◽ Documentar a API usando uma ferramenta de preferência.

Parte 4: Front-end Angular

  • ◽ Criar um aplicativo Angular que consuma as APIs desenvolvidas anteriormente.
  • ◽ Implementar funcionalidades para exibir, adicionar, editar e excluir produtos e categorias.
  • ◽ Criar uma interface de usuário simples, mas funcional.

Parte 5: Integração e Implantação

  • ◽ Implantar o aplicativo completo em um ambiente de teste com acesso público (por exemplo, Heroku, Azure App Service, AWS, Github ou serviço equivalente ou até local se acessível de fora da rede).
